Transfer to TCC from a Different School

To transfer to TCC follow these steps:

  1. Complete Application for Admission

    If you have never attended TCC or if you have not attended for 365 days or more, you have to apply for admission. You must provide proof of Texas residency.

    View Current Course Offerings

  2. Submit Proof of Meningitis Vaccination

    The meningitis vaccination is required for all students under the age of 22.

  3. Submit an Official Transcript from All Colleges/Universities Attended

    Request a transcript evaluation aftre TCC receives all of your transcript(s).

    1. Log in to Web Advisor
    2. Go to the Student Menu
    3. Go to the Academic Links section
    4. Sellect Transcript Evaluation Request

  4. Submit Texas Success Initiative (TSI) Documentation
